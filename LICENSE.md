# Licensing

This repository contains several kinds of work, each under an appropriate open-source license. This hybrid keeps the **hardware open and reciprocal**, makes the **code friction-free to reuse**, and asks only for **attribution on the writing**.

| Content | License | SPDX | Reference |
|---|---|---|---|
| Hardware & physical/electrical/mechanical designs | CERN Open Hardware License, Strongly Reciprocal, v2 | `CERN-OHL-S-2.0` | https://ohwr.org/project/cernohl/wikis/Documents/CERN-OHL-version-2 |
| Software & code (scripts, control software, models' formulas) | MIT | `MIT` | [LICENSES/MIT.txt](LICENSES/MIT.txt) |
| Documentation & written content (design docs, roadmap, diagrams) | Creative Commons Attribution 4.0 International | `CC-BY-4.0` | https://creativecommons.org/licenses/by/4.0/ |

## Plain language

- **You may** use, study, modify, and share this work, including commercially.
- **Hardware (CERN-OHL-S):** if you distribute a product based on these designs, you must make your modified design files available under the same license - the hardware stays open.
- **Code (MIT):** reuse freely; keep the copyright notice.
- **Docs (CC-BY):** reuse and adapt freely; credit "Divergent House Bus / Divergent Futures" and link back.
